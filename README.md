# notify-joke-extension

Extension that triggers notifications coming from a graphql server.

## Install

```bash
$ npm install
```

## Usage

Install Firefox if you haven't already.

```bash
$ npm start
```

## Development

```bash
$ npm run server
```

```bash
$ npm run dev chrome
$ npm run dev firefox
$ npm run dev opera
$ npm run dev edge
```

## Build

```bash
$ npm run server
```

```bash
$ npm run build chrome
$ npm run build firefox
$ npm run build opera
$ npm run build edge
```
